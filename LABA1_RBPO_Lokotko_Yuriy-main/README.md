Тема - Социальная сеть

Сущности
User: пользователь
Post: пост
Comment: комментарий
Like: лайк
Follow: подписка

Функции
Лента новостей (feed) — все посты от подписок пользователя
Посты пользователя — просмотр, редактирование, удаление
Комментарии к постам — добавление, редактирование, удаление
Лайки — один лайк на пару (User-Post), подсчёт количества лайков
Подписки — формирование ленты из постов подписок, управление подписками
